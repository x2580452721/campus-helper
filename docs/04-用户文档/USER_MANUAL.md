# 📖 Campus Helper 用户操作手册 (User Manual)

***

## 👨‍💼 第一部分：管理员操作手册

管理员后台是平台的控制中心，用于审核任务、管理用户和配置系统参数。

### 1. 登录后台

* **访问地址**: `http://localhost:5173` (本地) 或线上地址。

* **默认账号**: `admin@campus.com` / `admin123`。

### 2. 仪表盘 (Dashboard)

登录后默认进入仪表盘，可查看：

* **总用户数**: 平台注册用户总量。

* **总任务数**: 历史发布的任务总量。

* **待审核任务**: 当前需要处理的任务数量（重点关注）。

### 3. 任务审核 (Task Audit)

点击左侧菜单 **"任务审核"**。

* **列表展示**: 按时间倒序展示所有任务。

* **状态标签**:

  * `pending`: 待审核（橙色）。

  * `published`: 已发布/待接单（蓝色）。

* **操作**:

  * **通过**: 任务状态变为 `published`，将在小程序端对所有用户可见。

  * **拒绝**: 任务状态变为 `cancelled`，小程序端不可见。

### 4. 用户管理 (User Management)

点击左侧菜单 **"用户管理"**。

* **查看用户**: 查看昵称、学号、当前信用分。

* **调整信用分**:

  * 点击 **"调整信用"** 按钮。

  * 输入 **数值**（正数为加分，负数为扣分）和 **原因**。

  * 系统会自动记录变更日志。

* **封禁用户**: 点击 **"封禁"**，用户将无法登录小程序。再次点击可解封。

### 5. 系统设置 (System Config)

点击左侧菜单 **"系统设置"**。

* **min\_credit\_score**: 接单最低信用分门槛（默认 60）。

* **task\_timeout\_hours**: 任务自动超时时间。

* 点击 **"编辑"** 修改参数，点击 **"保存"** 即时生效。

## 📱 第二部分：小程序用户指南

### 1. 注册与认证

* **登录**: 打开小程序，点击“手机号一键登录”。

* **实名认证**: 首次登录需填写真实姓名和学号。认证通过后获得初始信用分（100分）。

### 2. 首页模式 (三态切换)

点击顶部状态栏切换模式：

* **💤 休息 (Off)**: 静默状态，不接收任何任务推送。

* **🚀 接单 (Active)**: 开启地图模式，显示附近可接任务。

* **⏳ 挂起 (Standby)**: 智能省电模式，仅在特定时间窗（如下课前）推送顺路单。

### 3. 发布任务

点击底部 **"发布"** 标签。

1. 填写任务标题、描述、选择类型（代取/求助）。
2. 点击 **"选择地点"**，在地图上选点。
3. 设置酬金。
4. 点击发布。任务需等待管理员审核通过后才会显示。

### 4. 抢单流程

1. 在首页地图点击 **任务标记 (Marker)**。
2. 进入详情页，查看任务要求和酬金。
3. 点击 **"立即抢单"**。

   * *注意：若信用分低于系统门槛（默认60分），将无法接单。*

### 5. 我的任务

点击底部 **"我的"** 标签 -> **"任务"**。

* **我发布的**: 查看自己发布的任务状态。

* **我接受的**: 查看已抢到的任务，完成后可联系发布者。

### 6. 信用分查询

点击底部 **"我的"** 标签 -> **"个人中心"**。

* 查看当前信用分仪表盘。

* 查看信用分变更历史记录（如违规扣分、奖励加分）。
