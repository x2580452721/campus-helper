# Campus Helper - 项目开发历程

## 📋 项目概述

Campus Helper（校园互助平台）是一个专为在校学生设计的任务互助平台，旨在通过数字化方式解决校园内学生之间的互助需求，如代取快递、帮买东西、学习辅导等。

## 🚀 关键技术突破

### 1. 地图架构创新
- **问题**：微信小程序原生地图功能受限，高德地图SDK在小程序中功能不完善
- **解决方案**：采用独立地图Web服务架构，通过WebView加载地图页面
- **优势**：Web端地图API更强大，交互更丰富，统一一套地图代码减少维护成本

### 2. 四端架构设计
- **管理后台**：React + Ant Design + Vite
- **小程序H5端**：Taro + React + NutUI
- **微信小程序端**：Taro + React + NutUI + WebView地图
- **独立地图Web服务**：纯HTML + JavaScript + 高德地图JS API

### 3. Android App打包
- 使用Capacitor框架将H5版本打包成原生Android App
- 支持热更新：App打开时自动加载最新版本
- 配置GitHub Actions自动构建流程

### 4. 性能优化
- 距离计算缓存
- 搜索防抖
- 快速排序算法
- 首页加载速度优化
