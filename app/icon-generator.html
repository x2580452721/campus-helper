📋 如何使用这些图标

1. 下载所有尺寸 - 点击 "📦 下载全部尺寸" 按钮

2. 准备Android项目 - 运行 cd app && npx cap add android（如果还没有Android项目）

3. 替换图标 - 将下载的PNG文件复制到对应的 android/app/src/main/res/mipmap-*/ 文件夹

4. 重新构建 - 运行 npm run build:android


快速方法： 使用 Capacitor Assets 工具

cd app && npm install @capacitor/assets --save-dev

npx capacitor-assets generate --android
